# Dynamic Programming

| Algorithm | File | Use Case |
|---|---|---|
| Longest Common Subsequence | `longest_common_subsequence.cpp` | Diff tools (git diff), DNA sequence comparison |
| Edit Distance (Levenshtein) | `edit_distance.cpp` | Spell checkers, fuzzy matching, plagiarism detection |
| 0/1 Knapsack | `knapsack_01.cpp` | Budget-constrained resource allocation |

Build and run individually with `g++ -std=c++17 -O2 <file>.cpp -o out && ./out`, or `make all` from the repo root.
